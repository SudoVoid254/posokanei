# ⚖️ PosoKanei - Price Comparison

Το **PosoKanei** είναι μια σύγχρονη, γρήγορη και πλήρως responsive single-page εφαρμογή (SPA) για τη σύγκριση τιμών προϊόντων από διάφορα καταστήματα λιανικής. Η εφαρμογή καταναλώνει τα δεδομένα από το επίσημο API του **posokanei.gov.gr** (μέσω custom Cloudflare Workers), με στόχο να επεκτείνει τις δυνατότητές του και να προσφέρει μια σαφώς πιο πλούσια, λειτουργική και προηγμένη εμπειρία χρήσης σε σχέση με την κρατική ιστοσελίδα.

---

## ✨ Λειτουργίες & Χαρακτηριστικά

- 🌲 **Δυναμικό Δέντρο Κατηγοριών:** Πλοήγηση σε πολλαπλά επίπεδα κατηγοριών με αυτόματη εμφάνιση του πλήθους των διαθέσιμων προϊόντων ανά ενότητα, προσφέροντας γρήγορη εύρεση του επιθυμητού τύπου προϊόντος.
- 🔍 **Έξυπνη Αναζήτηση & Φιλτράρισμα:**
  - Αναζήτηση σε πραγματικό χρόνο με χρήση τεχνικής *debounce* για ελαχιστοποίηση των περιττών κλήσεων στο API και βέλτιστη απόδοση.
  - Φιλτράρισμα ανά εύρος τιμής (Ελάχιστη / Μέγιστη) με δυναμικό υπολογισμό και εμφάνιση των ορίων της τρέχουσας κατηγορίας.
  - Φιλτράρισμα ανά κατασκευαστή (Brand).
  - Φιλτράρισμα ανά συγκεκριμένα καταστήματα λιανικής (Retailers).
- ⚖️ **Σύστημα Σύγκρισης Προϊόντων:** Δυνατότητα προσθήκης έως και 4 προϊόντων σε έναν διαδραστικό πίνακα σύγκρισης, όπου προβάλλονται συγκεντρωτικά ο κατασκευαστής, η ελάχιστη, η μέση και η μέγιστη τιμή, καθώς και ο αριθμός των διαθέσιμων καταστημάτων.
- ★ **Λίστα Αγαπημένων (Favorites):** Αποθήκευση προϊόντων για μελλοντική παρακολούθηση.
- 💾 **Persistence (Μνήμη Εφαρμογής):** Αυτόματη διατήρηση των επιλογών του χρήστη (όπως η λίστα αγαπημένων, τα προϊόντα προς σύγκριση και η προτίμηση θέματος) μέσω `localStorage`, ώστε να μην χάνονται κατά την ανανέωση της σελίδας.
- 🔗 **Διατήρηση Κατάστασης στο URL (Shareable Links):** Όλα τα ενεργά φίλτρα (αναζήτηση, κατηγορία, σελίδα, εύρος τιμών, brand, καταστήματα, ταξινόμηση) συγχρονίζονται σε πραγματικό χρόνο με τις παραμέτρους του URL. Αυτό επιτρέπει την άμεση κοινοποίηση ή αποθήκευση των ακριβών αποτελεσμάτων αναζήτησης.
- 📥 **Εξαγωγή Δεδομένων σε CSV:** Δυνατότητα άμεσης λήψης της λίστας αγαπημένων ή των αποτελεσμάτων του πίνακα σύγκρισης σε αρχεία μορφής **CSV** για περαιτέρω επεξεργασία.
- 🌙 **Δυναμικό Dark Mode:** Πλήρης υποστήριξη σκοτεινού θέματος (Dark Theme) με εύκολη εναλλαγή από ένα κουμπί.
- ⌨️ **Συντομεύσεις Πληκτρολογίου:** Keyboard shortcuts για προχωρημένους χρήστες που επιτρέπουν τη γρήγορη πλοήγηση χωρίς τη χρήση ποντικιού.

## ⌨️ Συντομεύσεις Πληκτρολογίου

Πατώντας το πλήκτρο `?` εντός της εφαρμογής, εμφανίζεται το παράθυρο βοήθειας με τις εξής λειτουργίες:

| Πλήκτρο | Λειτουργία |
| :---: | --- |
| `/` | Άμεση εστίαση (Focus) στο πεδίο αναζήτησης προϊόντων |
| `c` / `C` | Καθαρισμός όλων των ενεργών φίλτρων και επαναφορά της σελίδας |
| `n` | Μετάβαση στην επόμενη σελίδα αποτελεσμάτων (Next page) |
| `p` | Μετάβαση στην προηγούμενη σελίδα αποτελεσμάτων (Previous page) |
| `Esc` | Άμεσο κλείσιμο οποιουδήποτε ανοιχτού Modal παραθύρου |
| `?` | Εμφάνιση του αναδυόμενου παραθύρου βοήθειας |
